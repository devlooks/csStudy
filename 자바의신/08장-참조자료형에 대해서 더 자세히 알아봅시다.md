생성자는 반드시 만들어야 하나?

만약 매개변수가 있는 생성자를 만들고 매개변수가 없는 기본생성자를 호출하면 어떻게 되나요?

- 에러 발생, 
- 객체 생성시 기본 사용자가 생성 되긴 하지만, 다른 매개변수가 있는 생성자를 추가 했을시,  
  기본 생성자를 생성되지 않는다. 그러므로, 기본 생성자도 따로 선언해주어야 한다.

- this 키워드

- void
- void 메소드 강제 종료 명령어 return;

- static 메소드는 클래스 변수만 사용할 수 있다.
