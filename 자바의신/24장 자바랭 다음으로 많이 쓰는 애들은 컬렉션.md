- Key, value

Map에서 데이터를 저장하는 메소드는 무엇인가?

- put

Map에서 특정 키에 할당된 값을 가져오는 메소드는 무엇인가?

- get

- remove

Map에서 키의 목록을 가져오는 메소드는 무엇인가?

- keySet, entrySet

- size

- HashMap

- HashTable

- HashTable
